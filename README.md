## üí° Key Features

### Data Transformation & Cleaning
- Removed unnecessary columns (Row ID, Postal Code, Customer ID)
- Standardized data types for dates and numerical values
- Created calculated columns for Year, Month, and Quarter
- Built dedicated Calendar table for time intelligence

### DAX Measures Implemented
1. **Total Sales** - Revenue aggregation
2. **Total Profit** - Profit calculation
3. **Profit Margin** - Profitability percentage
4. **Total Orders** - Transaction count
5. **Average Order Value** - Per-transaction revenue
6. **Return Rate** - Return percentage tracking
7. **Sales Previous Year** - Year-over-year comparison
8. **YoY Growth** - Growth percentage calculation

### Interactive Visualizations
- **KPI Cards:** Total Sales, Total Profit, Profit Margin, Total Orders
- **Sales Trend Line Chart:** Monthly revenue trends
- **Top Products Bar Chart:** Top 10 products by profit
- **Category Donut Chart:** Sales distribution across categories
- **Geographic Map:** State-wise sales performance
- **YoY Comparison:** Current vs previous year analysis

### Dynamic Filtering (Slicers)
- Region filter
- Category filter
- Segment filter (Consumer/Corporate/Home Office)
- Date range filter
